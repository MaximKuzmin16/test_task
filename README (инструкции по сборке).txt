Компилировать файл task.cpp в командной строке с помощью команды:
g++ task.cpp -o task
Запустить скомпилированную программу с помощью команды:
task <test_file.txt>

<test_file.txt> - любой тестовый файл с входными данными из представленных

Все файлы должны находится в одной папке

Описание входных тестовых файлов:
test_file1.txt - взят из примера с содержанием самого тестового задания
test_file2.txt - строка с числом компьютеров в клубе не представляет целое положительное число
test_file3.txt - строка с временем открытия и закрытия клуба не подходит по формату
test_file4.txt - события идут не последовательно
test_file5.txt - номер стола больше количества столов указанных в конфигурации
test_file6.txt - имя клиента не подходит по формату
test_file7.txt - клиент пришёл, но он уже в клубе
test_file8.txt - клиент сел за стол, который занят; клиент сел за стол, но не находится в клубе
